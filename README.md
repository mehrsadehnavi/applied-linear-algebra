# Applied Linear Algebra

## Overview
This project is a hands-on implementation of core concepts in applied linear algebra using Python and NumPy. It covers vector and matrix operations, linear transformations, and practical applications such as image processing and geometric transformations.

---

## Topics Covered
- NumPy fundamentals
- Arrays and matrix structures
- Array arithmetic and vectorization
- Dot product and matrix multiplication
- NumPy data types
- Array generation functions
- Array slicing and subarrays
- Elementary row operations (row swap, scaling, row addition)
- Linear transformations
- Geometric transformations (rotation, scaling, shear)
- Image processing (color grading and grayscale conversion)

---

## Key Implementations

### 1. Vector & Matrix Operations
- Element-wise operations
- Inner (dot) product
- Matrix multiplication
- Broadcasting and vectorization

### 2. Linear Algebra Operations
- Implementation of elementary row operations:
  - Row swap
  - Row scaling
  - Row addition
- Matrix-based transformations

### 3. Image Processing with Linear Algebra
- RGB color transformations using matrices
- Grayscale conversion:
  - Average method
  - Luminosity method
- Pixel-wise transformation using linear mappings

### 4. Geometric Transformations
- 2D transformations using matrices:
  - Rotation
  - Scaling
  - Shearing
- Visualization of transformations using matplotlib
